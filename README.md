## Тест-кейс 9: Попарное тестирование фильтров на странице Офисы и банкоматы

https://www.mtsbank.ru/

Шаги:
1. Перейти по ссылке Офисы и банкоматы

Ожидаемый результат: отображается страница Адреса и график работы всех офисов и банкоматов МТС Банка

2. Нажать на элемент фильтров

Ожидаемый результат: Отображаются флажки с фильтрами. Отмечены следующие флажки: работает сейчас и все фильтры тип объекта

Ожидаемый результат: фильтры скрыты

5. нажать на кнопку списком

Ожидаемый результат: отображаются только Офисы МТС Банка

6. Аналогично шагу 3, применить действие для остальных фильтров тип объекта

Ожидаемый результат: выбрана пара фильтров: работает сейчас, тип объекта

7. Повторить шаги 4 и 5

Ожидаемый результат: отображаются только объекты, согласно выбранной паре фильтров
